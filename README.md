Ce projet est une application de démonstration en Java simulant un système de gestion de comptes bancaires.

##  Fonctionnalités

- Héritage de classe : Centralisation des données communes dans une classe parente (BankAccount) réutilisée par trois types de comptes spécifiques.
- Gestion des comptes courants (CheckingAccount) avec ajout d'une limite de découvert
- Gestion des comptes d'épargne(SavingsAccount)
- Gestion des certificats de dépôt (COD) avec un identifiant de produit unique
- Encapsulation stricte: Tous les attributs sont protégés et accessibles uniquement via des getters et setters
